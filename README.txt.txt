The repo contains:

1- Train_Test_Mean_Std_data.csv : dataset merging train and test data fromt he UCI HAR Dataset and showing only mean and standard deviation measurements

2- Train_Test_Mean_Std_data_summary.csv: summary of above dataset for each (subject,activity) pair showing averages of variables in above dataset

3- Codebook.md: description of the two above data sets and their variables

4- run_analysis.R: R script used to generate the above described data set from the original UCI HAR Dataset
